Titanic Survival Prediction

Project Overview
This project builds an end-to-end machine learning classification model to predict whether a passenger survived the Titanic disaster based on demographic and travel-related features.
The project demonstrates data preprocessing, exploratory data analysis (EDA), feature engineering, and supervised learning techniques.

Problem Statement
Given passenger data such as age, gender, class, and fare, predict whether the passenger survived.
This is a binary classification problem.

Dataset
This project uses the Titanic dataset from Kaggle:
https://www.kaggle.com/datasets/yasserh/titanic-dataset
Download the dataset and place titanic-dataset.csv inside the data/ folder before running the notebook.
